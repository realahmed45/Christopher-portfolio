# Images Folder

Place all your project images here.

## Naming Convention

Use descriptive, lowercase names with hyphens:

**Examples:**
- `bishop-ranch-layout-1.jpg`
- `bishop-ranch-layout-2.jpg`
- `csag-banner-before.jpg`
- `csag-banner-after.jpg`
- `csag-logo-variations.png`
- `filam-logo-color.png`
- `filam-logo-bw.png`
- `filam-logo-animated.gif`
- `undo-mindset-social-1.jpg`
- `undo-mindset-social-2.jpg`
- `undo-mindset-social-3.jpg`

## File Formats

- **JPG**: For photographs and complex images
- **PNG**: For graphics with transparency, logos
- **GIF**: For simple animations

## Optimization

Keep file sizes small for fast loading:
- Target: Under 500KB per image
- Use online tools like TinyPNG or Squoosh
- Recommended dimensions: 1200-1600px wide for project images

## Organization

You can create subfolders if you have many images:

```
images/
├── bishop-ranch/
│   ├── layout-1.jpg
│   └── layout-2.jpg
├── csag/
│   ├── banner-v1.jpg
│   └── logo-variations.png
└── filam/
    ├── logo-color.png
    └── logo-bw.png
```

If you use subfolders, update image paths in HTML:
```html
<img src="../images/bishop-ranch/layout-1.jpg" alt="Layout 1">
```
